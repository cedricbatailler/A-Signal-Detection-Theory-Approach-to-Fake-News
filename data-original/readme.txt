2019-09-27 data-raw
The folder contains raw data used for the analysis. Data used come from two 
papers:
- 10.1016/j.cognition.2018.06.011
- 10.1037/xge0000465

Data sets were retrieve from the osf repo associated to the papers. These data
sets are stored in a folder named according to the id of the osf repo. An 
additional file was ussed for the analyses 
(20190426 - grpennycook@gmail.com - Counterbalancing v2.xlsx), and was obtained 
through mail correspondance with Gordon Pennycook.
